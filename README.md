# Antimatter [media art] — Program Archive (1998–2024)

A searchable archive of every work programmed at the **Antimatter [media art]**
festival (Victoria, BC) from 1998 to 2024, mined from the festival's public
program guides. Full-text search + faceted filters (year, type, premiere, venue,
country, artist region, genre, medium, runtime), an artist index, and per-work
detail with links to the source program PDFs.

**Live:** https://bturep.github.io/antimatter-archive/

Single self-contained page (`index.html`). Source guides: https://antimatter.squarespace.com/previous
Descriptions are quoted from the catalogues. No personal contact data is included.
A research/reference project; not affiliated with the festival.
